An illustration of a typical ANN model. Each of the eight variables in the first layer (labelled as X1-X8) are input layer. The output layer is shown in the far right (labelled as ‘y’). H1 through H10 is hidden layer, although a more complex ANN model can have many hidden layers, in the demonstration is labelled as B1 and B2 are bias layers that apply constant values to the nodes, similar to intercept terms in a regression model.
